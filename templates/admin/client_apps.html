üìö API Usage Guide

1. Get API Token:

POST /api/v1/auth/token
Content-Type: application/x-www-form-urlencoded

app_id=your_app_id&app_secret=your_app_secret&expires_in=3600

2. Use Token in API Calls:

GET /api/v1/users
Authorization: Bearer your_jwt_token

3. Token expires in 1 hour by default. Request new tokens as needed.
